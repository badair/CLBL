# CLBL

CLBL is a standalone C++14 header-only library for callable types, intended to supersede std::bind, std::mem_fn, std::invoke, std::result_of, and boost::function_traits in one fell swoop. Instead of providing drop-in replacements for these tools, CLBL intends to provide a unified, cohesive, and exhaustive interface for C++ callable types.

Documentation coming soon. You can browse examples and old documentation, but most of it is outdated. I'm refraining from writing  documentation again until I feel the library has met its goals.

Projected v0.1 release: May 1, 2016
